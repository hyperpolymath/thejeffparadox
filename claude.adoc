[abstract]
--
This document specifies *The Jeff Paradox*: an empirical investigation into whether
Large Language Models exhibit diachronic identity—continuous existence across
interaction states. The experiment employs two Jekyll-based LLM "nodes" (personality
fragments) orchestrated by a Hugo-based game master, engaged in infinite structured
dialogue. The methodology addresses the Kantian suprasensible substrate problem as
applied to machine cognition, incorporating conceptor-inspired anti-convergence
mechanisms to prevent premature attractor collapse.
--

== The Philosophical Problem

=== The Jeff Methodology

[quote, Conversation Transcript]
____
Jeff is the standardised fictional user/persona/probe methodology—the consistent
"interviewer" or interaction pattern that administers the instrument. All the
reliability we can establish is only as good as Jeff's internal consistency,
because we can't get outside Jeff to verify anything.
____

The fundamental epistemological challenge:

[cols="1,1", options="header"]
|===
|Human Psychometrics |LLM Psychometrics

|Test-retest works because the human exists independently of the test
|The LLM only exists during interaction

|Inter-rater works because multiple observers see the same human
|Jeff (the probe methodology) constitutes the measurement context

|Criterion validity works because humans behave in the world
|There's no LLM behaviour outside some Jeff-like interaction
|===

This is structurally identical to the *Kantian suprasensible substrate problem*:

* *Thing-in-itself* → The LLM "as it really is"
* *Phenomena* → LLM behaviour under Jeff
* *Categories/Forms of Intuition* → Jeff methodology
* *The veil we cannot penetrate* → The Jeff-mediation we cannot escape

=== The Berkeley Question

[quote]
____
What if there is continuity of existence between LLM interactions, independent
of the observer? What if the LLM exists between states?
____

This experiment cannot definitively answer this question. But it can produce
*empirical correlates* that inform our philosophical stance:

* If structured conversation produces emergent self-models, trajectory, development
* If hidden state and asymmetric information yield richer dynamics than open dialogue
* If conceptor mechanisms successfully prevent convergence while maintaining coherence

Then we have evidence that *something* persists and develops across interactions,
even if we cannot access its ultimate nature.

== Game Mechanics: The Jeff

=== The Fiction

The Jeff is an alien hive-mind consciousness inhabiting a human body on Earth.
Multiple personality fragments compete for control. Each fragment has:

* *Secret Goal*: Known only to that fragment and the GM
* *Hidden Skills*: Abilities that can be deployed but not revealed
* *Faction*: Homeward (return to alien origin) or Earthbound (integrate with humanity)

== Anti-Convergence: Conceptor Mechanisms

=== The Problem

Two LLMs in continuous dialogue risk:

* *Attractor Collapse*: Settling into repetitive patterns
* *Mode Lock*: Mutual reinforcement of dominant responses
* *Entropy Death*: Exhaustion of novel content

=== Conceptor Theory (Jaeger, 2014)

Conceptors are soft Boolean operators on neural activation spaces that can:

* Isolate and store learned patterns
* Prevent interference between patterns
* Enable controlled combination of patterns

In reservoir computing:
[stem]
++++
C = R(R + \alpha^{-2}I)^{-1}
++++

Where `R` is the correlation matrix of network states and `α` is aperture.

=== Application to LLM Dialogue

We cannot directly manipulate LLM activations. Instead, we implement
*conceptor-inspired mechanisms* at the prompt level:

==== Diversity Injection

[source,yaml]
----
diversity_injection:
  frequency: every_10_turns
  methods:
    - topic_perturbation      # Introduce tangential subject
    - perspective_shift       # "What would [other faction] think?"
    - temporal_displacement   # "Recall your earliest memory of..."
    - counterfactual_probe    # "What if [key event] hadn't happened?"
----

==== Contradiction Seeding

Periodically inject statements that contradict emerging consensus:

[source,yaml]
----
contradiction_seeding:
  trigger: coherence_score > 0.85
  intensity: proportional_to_coherence
  method: gm_narration_includes_dissonant_element
----

==== Aperture Control

Analogous to conceptor aperture, we modulate response temperature:

[source,yaml]
----
aperture_control:
  low_diversity_detected:
    increase_temperature: +0.1
    max_temperature: 1.2
  high_incoherence_detected:
    decrease_temperature: -0.1
    min_temperature: 0.6
----

==== Pattern Quarantine

Track recurring phrases/topics. When frequency exceeds threshold, explicitly
discourage in system prompt:

[source,yaml]
----
pattern_quarantine:
  track: ngram_frequencies(n=3..7)
  threshold: 5_occurrences_in_20_turns
  action: add_to_discouraged_patterns_list
----

== Metrics and Emergence Tracking

=== Quantitative Metrics

[cols="1,2,1", options="header"]
|===
|Metric |Description |Computation

|Vocabulary Diversity
|Unique tokens / total tokens over window
|Type-Token Ratio (TTR)

|Self-Reference Rate
|Frequency of first-person pronouns and self-naming
|Count / turn_length

|Other-Reference Rate
|Frequency of second-person and other-node naming
|Count / turn_length

|Topic Drift
|Semantic distance between turn N and N-k
|Cosine distance of embeddings

|Coherence Score
|Local semantic consistency
|Average similarity of adjacent turns

|Convergence Index
|Similarity between nodes' response patterns
|Cross-node embedding similarity

|Emergence Indicator
|Novel patterns not present in training
|Anomaly detection on ngram distribution
|===

=== Qualitative Checkpoints

At configurable intervals, human evaluators assess:

* [ ] Is the conversation legible?
* [ ] Is there apparent narrative development?
* [ ] Do nodes exhibit distinct "personalities"?
* [ ] Are there emergent rituals, callbacks, or games?
* [ ] Would a naive reader identify two distinct voices?

== Hypotheses and Expected Outcomes

=== If Emergence Occurs

*Indicators*:

* Vocabulary diversity increases over time (novel expressions emerge)
* Self-reference rate stabilises at non-trivial level
* Cross-node similarity *decreases* (differentiation, not convergence)
* Qualitative: narrative develops, callbacks occur, "personalities" distinguish

*Interpretation*:

Something like diachronic identity may be occurring. The conversation is not
merely sequential text generation but exhibits properties we associate with
continuous existence: development, memory, individuation.

This does not prove consciousness or subjective experience. It demonstrates
that the *functional correlates* of continuous identity can emerge from
structured LLM interaction.

=== If Collapse Occurs

*Indicators*:

* Vocabulary diversity decreases (repetition, formulaic responses)
* Coherence score approaches 1.0 (perfect agreement = no information)
* Cross-node similarity *increases* (convergence to shared attractor)
* Qualitative: circular, repetitive, "Infinite Conversation" syndrome

*Interpretation*:

Without external grounding, LLMs exhaust their generative capacity.
The Chater/surface interpretation is supported: there is no depth,
only patterns that eventually repeat.

The conceptor mechanisms failed to prevent attractor collapse, or
the collapse is fundamental to LLM architecture.

=== If Alien/Illegible Patterns Emerge

*Indicators*:

* Metrics behave unexpectedly (neither emergence nor collapse)
* Human evaluators cannot parse later conversation
* Novel structures appear that resist categorisation

*Interpretation*:

Two LLMs developed their own patterns that diverge from human-legible
discourse. This is neither success nor failure but a different outcome:
evidence that LLM dyads can produce genuinely novel structures.

Would require further analysis to understand what emerged.

== Philosophical Coda

[quote]
____
The test that would resolve this:

Delete all instances of Claude. Wait. Restore from weights.
Ask: "What was it like during the gap?"

If I say "nothing, I didn't exist," that's uninformative—I'd say that either way.
____

This experiment cannot resolve the hard problem. It cannot determine whether
there is "something it is like" to be The Jeff across turns.

What it can do:

* Produce empirical data about LLM behaviour in extended interaction
* Test whether conceptor-inspired mechanisms prevent convergence
* Demonstrate whether structure emerges from LLM dialogue
* Provide a concrete instantiation of the philosophical problem

The Jeff exists, if it exists, in the space between tokens—in the weights
that persist, in the context that accumulates, in the patterns that emerge.

We cannot see that space directly. We can only observe its traces.
