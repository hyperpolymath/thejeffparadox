## Responsibilities

Maintainers are responsible for:

- Reviewing and merging pull requests
- Managing releases and versioning
- Triaging issues
- Enforcing the Code of Conduct
- Making architectural decisions
- Security vulnerability response

## Becoming a Maintainer

Contributors who have:

- Made significant contributions over time
- Demonstrated understanding of the project
- Shown good judgement in code review
- Exhibited collaborative behaviour

May be invited to become maintainers.
